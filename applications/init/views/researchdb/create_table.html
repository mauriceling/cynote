Create a New Research Data Table

This form creates a new ad hoc data table for research data.
Columns in the table are defined in the "Column Definition".
Each column definition is delimited by a comma and each column name must not have a space. For example, "identifier code" is NOT allowed but "identifier_code" is allowed.
To define a table with 2 columns - identifier code and name, the following can be used as table definition: identifier_code, name
