Generic and Common Actions

          [ Create New Notebook Entry ] Create a new entry in one of the existing notebooks

          [ Create New Notebook ] Create a new notebook to hold entries as every entry must be part of a notebook

          [ List Entries by Notebook ] Show a list of entries for a particular notebook (Table of Contents)

          [ Notarize An Entry ] Signing off a single entry as an individual

          [ Backup Notebooks ] Allows for backup of notebooks as separate files and to remote FTP sites

          [ Archive Notebook(s) ] Set notebook(s) for archiving

          [ Un-Archive Notebook(s) ] Release previously archived notebook(s) for use

Bioinformatics Analysis Actions

          [ Orphaned Analysis Results ] List of unsaved analysis results (once an analysis result is saved as an entry, it will disappear from here)

          [ Analyze Primers ] Analyze a single primer or a pair of primers for annealing temperature and secondary structures

          [ Primer Design ] Designing primer pairs from given DNA sequence

          [ Search PubMed ] Performs literature search on NCBI PubMed

          [ Basic Sequence Utilities ] Given a sequence - Transcribes, Reverse transcribes, Translates

          [ Protein Analysis ] Analyse amino acid sequence for amino acid distribution, pI, MW, aromaticity, instability, flexibility, and secondary
                                                  structure fractions.

          [ Similarity Search ] Identify given sequences using different BLAST tools

          [ Phylogenetics ] Multiple sequence alignment and phylogenetic analyses

          [ Restriction Mapping ] Maps out the restriction endonuclease recognition sites of a sequence
